Back To Mac

October 16, 2010 at 05:26 PM | categories: Mac, Others | View Comments

据苹果公司透露，苹果将于美国西部时间10月20日上午10点（北京时间10月21日凌晨）举办今年最后一场发布会。会议主题为“Back to the Mac”。 从各地流传的新闻来看,新的发布会将可能放出下面几款产品:

苹果下周推出 11.6 英寸 MacBook Air(Apple4US)

AppleInsider 报道称，他们已经从多个独立的消息源得到证实，苹果将在下周三的媒体发布会上推出重量更轻、屏幕 11.6 英寸、外观彻底重新设计的全新 MacBook Air ，再次对超便携笔记本市场发起冲击。 据消息源称，第一款可以确定配备了 11.6 英寸显示屏的 MacBook Air 已经从台湾的工厂装运发货至少已经一周，在「Back to the Mac」发布会举行后不久即可上市。 另一位消息源称，新款 MacBook Air 外形与当前的约 13.3 英寸的版本有显著的变化。 此外，新款 MacBook Air 存储设备也有重大变化。据其称，新款 MacBook Air 除去了现有的 HDD 硬盘和 SSD 固态硬盘选项，取而代之的是一种称之为的「SSD 卡」的存储原件。在结构上它没有传统驱动附件，更像是一个内存条，但用户并不能轻易将其替换。 让新的 MacBook Air 采用类似于闪存存储可以使其快速启动，或从睡眠中快速恢复，甚至做到类似与 iPad 或 iPhone 的「即开即用」，。 第三位据称见过新款原型的消息源表示，在外形设计新款的边缘线条更为明晰，而当前的 MacBook Air 边缘是有弧度的曲面。新的设计让 MacBook Air 看上去更加锐利。

WP-Syntax - Code HighLighting 代码高亮插件 for Wordpress

September 26, 2010 at 04:33 PM | categories: PHP, Wordpress, Software | View Comments

WP-Syntax provides clean syntax highlighting for embedding source code within pages or posts. WP-Syntax provides clean syntax highlighting using GeSHi -- supporting a wide range of popular languages. It supports highlighting with or without line numbers and maintains formatting while copying snippets of code from the browser. It avoids conflicts with other 3rd party plugins by running an early pre-filter and a late post-filter that substitutes and pulls the code snippets out first and then pushes them back in with highlighting at the end. The result is source code formatted and highlighted the way you intended. This plugin was originally written for use with EmacsBlog. To see it in action, scroll through this particular post or visit the screenshots. Usage, Supported Languages, Styling Guidelines, and Release Notes are availabe in the Other Notes section.

Wrap code blocks with <pre lang="LANGUAGE" line="1"> and </pre> where LANGUAGE is a GeSHi supported language syntax.
